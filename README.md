# Practica-de-Redes
En esta practica hay un script de menu donde se pueden realizar distintas acciones (muy basicas) en Linux (ya sea alguna distro Debian o Ubuntu). Son básicas, como bloquear el acceso a una IP o rango IP o Bloquear redes sociales (por mencionar). Podría mejorarse?
