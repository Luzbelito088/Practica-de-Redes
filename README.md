# Practica-de-Redes
En esta practica hay un script de menu donde se pueden realizar distintas acciones (muy basicas) en Linux (ya sea alguna distro Debian o Ubuntu). Son básicas, como bloquear el acceso a una IP o rango IP o Bloquear redes sociales (por mencionar). Podría mejorarse? Seguramente, pero esta era una tarea simple que decidí llevarla un porquito mas allá al colocarle un menú de opciones y llamar los scripts desde allí.
El menu basicamente es: 
1) Asegurar y verificar iptables
2) 2) Bloquear todo el trafico
3) Permitir trafico desde host compañero
4) Bloquear trafico desde rango IP
5) Bloquear trafico hacia Gloogle
6) Bloquear acceso al DNS
7) Permitir trafico desde Clarin
8) Permitir trafico hacia UNSJ
9) Denegar acceso a redes sociales
10) Denegar acceso POP mail
11) Hacer reglas persistentes
